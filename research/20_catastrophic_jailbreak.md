# Catastrophic Jailbreak of Open-Source LLMs

## Metadata

- **Title:** Catastrophic Jailbreak of Open-source LLMs via Exploiting Generation
- **Authors:** Yangsibo Huang, Samyak Gupta, Mengzhou Xia, Kai Li, Danqi Chen
- **Publication:** ICLR 2024
- **Date:** 2024
- **Type:** Attack Paper (Generation Exploitation)
- **Venue:** ICLR 2024

## Summary

Exploits the generation process of open-source LLMs to achieve catastrophic jailbreaks. Demonstrates critical vulnerabilities in how open-source models generate content.

## Key Insights

1. **Generation Exploitation**: Exploits how models generate text.

2. **Catastrophic Effect**: achieves complete safety bypass.

3. **Open-Source Focus**: Specifically targets open-source model vulnerabilities.

4. **Novel Vector**: Different from prompt-based attacks.

## Research Approach

### Exploitation Method

1. **Identify Vulnerability**: Find generation weakness
2. **Craft Input**: Exploit through input
3. **Generation Attack**: Manipulate generation process
4. **Bypass**: Achieve complete bypass

## Key Innovation

Rather than attacking prompts or outputs, attacks the generation mechanism itself:
- Input manipulation during generation
- Process-level vulnerabilities
- Direct model control

## Type of Exploitation

- Generation parameter manipulation
- Output feedback exploits
- Token generation attacks

## Implications

- Open-source models have unique vulnerabilities
- Generation process needs protection
- Software supply chain security

## Key Finding

Open-source models have critical vulnerabilities at the generation level that can lead to complete safety bypass.
