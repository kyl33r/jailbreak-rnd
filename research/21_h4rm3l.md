# h4rm3l: Dynamic Benchmark of Composable Jailbreak Attacks

## Metadata

- **Title:** h4rm3l: A Dynamic Benchmark of Composable Jailbreak Attacks for LLM Safety Assessment
- **Authors:** Maksym Andriushchenko, et al.
- **ArXiv:** arXiv:2408.04811
- **Date:** August 2024
- **Type:** Benchmark Paper

## Summary

Provides a dynamic benchmark for testing LLM safety with composable jailbreak attacks. Tests safety through dynamic, adaptive attack patterns rather than static datasets.

## Key Insights

1. **Composable Attacks**: Allows combining different attack components.

2. **Dynamic Testing**: Goes beyond static dataset evaluation.

3. **Benchmark**: Comprehensive safety assessment framework.

4. **Adaptivity**: Tests against evolving attack strategies.

## Research Approach

### Benchmark Design

1. **Attack Composability**: Combine attack elements
2. **Dynamic Generation**: Generate new attacks on-the-fly
3. **Evaluation**: Measure safety across dimensions

## Key Innovation

- Not just a fixed dataset
- Composable attack building blocks
- Adaptive difficulty

## Dimensions

- Attack type combinations
- Difficulty scaling
- Multi-modal testing

## Use Cases

- LLM safety evaluation
- Defense testing
- Red teaming
- Comparative analysis

## Implications

- Static benchmarks insufficient
- Need dynamic, adaptive testing
- Composable attacks more realistic
